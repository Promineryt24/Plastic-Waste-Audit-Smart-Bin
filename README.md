# ♻️ Plastic Waste Audit Smart Bin

> A semi-automated waste bin that differentiates between recyclable and non-recyclable plastic waste, while keeping an audit log of disposal habits.

---

## 📖 Overview

The **Plastic Waste Audit Smart Bin** is designed to tackle the growing issue of plastic waste mismanagement. By automating the sorting process, this project helps ensure that recyclable plastics are properly separated from non-recyclables. Additionally, it features an audit logging system that records data for further analysis, making it an excellent tool for environmental monitoring and waste management research.

## ✨ Key Features

* **Semi-Automated Sorting:** Intelligently differentiates between recyclable and non-recyclable plastic items.
* **Audit Logging:** Automatically logs waste sorting data into a CSV file for auditing and analysis.
* **Python-Powered:** Built using Python, making it accessible and easy to modify or expand.

---

## 🗂️ Repository Structure

| File Name | Description |
| :--- | :--- |
| `app smart bin 5.py` | The main Python application script that runs the sorting and logging logic. |
| `app_smart_bin_audit.csv` | The generated audit log that records the history and types of processed waste. |
